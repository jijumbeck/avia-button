
# Проект-шаблон сборки для React проекта на Typescript

## Описание

Сборка включает в себя:

* Запуск проекта в браузере;
* Резолв CSS файлов;
* Резолв Typescript;

## Запуск

```bash

# Запуск приложения в браузере.

$  npm  run  start

# Build в development mode.

$  npm  run  build:dev

# Build в production mode.

$  npm  run  start:prod

```
